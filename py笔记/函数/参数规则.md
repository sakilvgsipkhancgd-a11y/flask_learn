1.必须参数
表示必须传入的参数，调用函数时必须传入相应的参数。
并且位置数量要一致
def add(a,b):
    return a+b
add(1,2) # 3
add(1)会报错，缺少参数
2.关键字参数
可以通过参数名来传入参数，关键字参数在函数调用时可以不按顺序传入参数，也可以只传入部分参数。
def add(a,b):
    return a+b
add(b=2,a=1) # 3
add(1,2) # 3结果一样，关键字参数可以避免因为参数顺序导致的错误
也可以和位置参数混合使用，但是位置参数必须在前
3.默认参数
函数可以设置默认参数，当函数调用时没有传入相应的参数时，会使用默认参数。
def add(a,b=1):
    return a+b
add(1)#2这里传入了位置参数，b使用默认参数1
add(1,2)#3这里不仅传入了位置参数，还修改了默认参数
